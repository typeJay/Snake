# Snake
Snake Game in HTML by an open-source model running on my PC

Features of this Snake Game:
Complete Game Functionality:

Snake movement with arrow keys
Food collection and score tracking
Collision detection (walls and self)
Game over screen with restart option
Visual Design:

Attractive gradient background
Responsive game board with grid
Colorful snake with head differentiation
Shiny food items
Clean, modern UI with game controls
Game Controls:

Start/Restart button
Pause/Resume button
Reset button
Mobile touch controls for smaller screens
Additional Features:

Score and high score tracking (saved in local storage)
Increasing difficulty (snake speeds up as you eat more food)
Responsive design that works on both desktop and mobile
Game instructions for new players
Simply copy and paste this code into an HTML file and open it in any modern browser to play! Use the arrow keys to control the snake's direction.
